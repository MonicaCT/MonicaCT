## About

My portfolio is organized by capabilities rather than employers. Each project is classified by:

- Domain: the policy, economic, or business problem being studied.
- Method: the analytical approach used to answer the question.
- Tools: the reproducible workflow used to produce evidence.

This structure makes the portfolio easy to read from different angles: economics, public policy, research methods, analytics, business intelligence, and data science.

## Portfolio Logic

Every project is designed to be readable through three questions:

1. What problem does it study?
2. What method does it use?
3. What tools make the work reproducible?

That is the organizing principle behind this GitHub profile.
