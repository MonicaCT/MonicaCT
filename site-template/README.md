# Portfolio Project Website Template

Reusable static website template for Monica Cueto Tapia's research and analytics repositories.

## Purpose

This template provides a shared visual and navigational standard for repository-level GitHub Pages sites. It is designed for existing analytical products such as dashboards, reports, figures, tables, methodological notes and replication documentation.

## Files

```text
site-template/
  index.template.html
  project-content.schema.json
  IMPLEMENTATION_GUIDE.md
  assets/
    css/site.css
    js/site.js
```

## Required Content

Populate a repository-level `project-content.json` following `project-content.schema.json`, then replace the placeholders in `index.template.html` or generate an `index.html` from the template.

The template supports:

- Hero
- Executive Summary
- Key Metrics
- Key Findings
- Dashboard
- Main Figures
- Executive Tables
- Data Sources
- Analytical Workflow
- Methodology
- Paper and Reports
- Limitations
- Reproducibility
- Citation
- Author
- Related Projects
- Back to Portfolio

## Privacy Rule

Do not publish restricted data, local paths, personal identifiers, telephone numbers, credentials, tokens or private raw microdata. Link only to artifacts already intended for public use.
